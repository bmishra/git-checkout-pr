# git-checkout-pr

git-checkout-pr fetches a pull request from GitHub and checks it out into a local branch.

## Installation

```
$ go get github.com/broady/git-checkout-pr
```

## Usage

```
git-checkout-pr [-branch branchname] [-v] url-to-pull-request
```
